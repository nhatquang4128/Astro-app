## Run Locally

Clone the project

```bash
  git clone https://github.com/nhatquang4128/Astro-app.git
```

Go to the project directory

```bash
  cd Astro-app
```

Install dependencies

```bash
  yarn
```

Start the server

```bash
  yarn start
```
